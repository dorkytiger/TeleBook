# TeleBook 3.0：全新升级的跨平台图书管理与阅读应用

**TeleBook** 是一个使用 **Flutter** 开发的功能强大的跨平台图书解析、管理与阅读应用。支持多种导入方式、智能书库管理、个性化阅读体验，让数字阅读变得前所未有的便捷和舒适。

## ✨ 3.0 版本重大更新

### 📚 全新书库管理系统
- **🗂️ 收藏夹功能**：创建自定义收藏夹，支持设置图标和颜色，智能分类管理书籍
- **⭐ 星标系统**：为书籍添加彩色标签，快速标识和筛选重要内容
- **📊 布局切换**：支持列表视图和网格视图两种显示模式，个性化书架体验
- **⚡ 批量操作**：批量选择、移动、删除、导出书籍，高效管理大量内容
- **🔍 智能筛选**：按收藏夹、标签、搜索关键词快速定位目标书籍

### 📖 个性化阅读体验
- **📍 阅读进度记录**：自动保存每本书的阅读位置，断点续读无缝衔接
- **🔄 多种阅读方向**：支持从左往右、从右往左（适合漫画）、从上往下（长条模式）三种阅读方式
- **⚙️ 阅读设置**：点击设置按钮即可快速切换阅读方向，设置自动保存
- **📊 智能进度条**：根据阅读方向显示不同样式的进度指示器，上下阅读显示百分比进度

### 📝 书籍编辑增强
- **✏️ 即时重命名**：在编辑页面快速重命名书籍，立即生效无需保存
- **🗑️ 智能删除**：删除图片时自动清理物理文件和数据库记录，防止存储空间浪费
- **📄 实时同步**：所有编辑操作立即同步到数据库，确保数据一致性

### 🎨 界面与性能优化
- **🌙 深色模式适配**：完整支持系统深色模式，护眼舒适的夜间阅读
- **⚡ 性能优化**：优化内存使用、加载速度和响应性能
- **🐛 问题修复**：修复已知bug，提升应用稳定性
- **💫 UI美化**：界面细节优化，更加现代化的设计语言

### 🌐 多元化内容导入

#### WebView 网页解析
- 内置 WebView 浏览器，直接加载网页内容
- 智能提取页面中的所有图片
- 支持 Telegraph 及其他图片类网页
- 实时预览，选择性下载

#### 单个压缩包导入
- 支持 ZIP、CBZ、RAR 等多种压缩格式
- 自动解压并识别图片文件
- 快速建立书库
- 统一文件命名（8位数字前导零格式）

#### 批量压缩包导入
- 选择文件夹，自动扫描所有 ZIP/CBZ 文件
- 并行解压，后台处理不阻塞 UI
- 可编辑每个压缩包的文件列表（排序、删除）
- 批量保存，一键导入多本书籍

#### PDF文件导入
- 直接导入PDF文件并转换为图片格式
- 支持高质量图片渲染
- 自动分页处理

#### 图片文件夹导入
- 直接导入包含图片的文件夹
- 支持单个文件夹和批量文件夹导入
- 自动按文件名排序

### 📤 强大的导出功能

- **单个书籍导出**：导出为标准ZIP压缩包
- **批量导出**：选择多本书籍一键导出
- **后台处理**：使用Isolate技术，导出过程不影响UI流畅度
- **智能命名**：自动使用书籍名称作为文件名

```bash

# 安装依赖
flutter pub get

# 编译
flutter build apk        # Android
flutter build ios        # iOS
flutter build windows    # Windows
flutter build macos      # macOS
flutter build linux      # Linux
```

## 🚀 快速开始

### 📖 导入你的第一本书

#### 方式一：通过网页链接解析
1. **复制书籍链接** - 在 Telegraph 或其他网页找到想要的书籍，复制链接地址
2. **打开导入界面** - 点击右上角的 ➕ 号
3. **选择网页数据源** - 选择"网页"，粘贴链接到"网址"字段
4. **解析并预览** - 应用自动加载网页并提取图片，实时预览
5. **添加到下载队列** - 点击"保存"按钮，图片将被添加到下载队列
6. **保存书籍** - 下载完成后，点击"保存书籍"将书籍添加到书架

#### 方式二：通过压缩包导入
1. **准备压缩包** - 将图片文件打包成 ZIP、CBZ 或其他压缩格式
2. **选择压缩包数据源** - 点击 ➕ 号，选择"压缩包"
3. **选择文件** - 点击"选择文件"，从设备中选择压缩包
4. **自动解压** - 应用自动解压并识别图片，点击"保存"添加到书架

#### 方式三：批量压缩包导入
1. **准备文件夹** - 将多个 ZIP/CBZ 文件放在同一文件夹
2. **选择批量数据源** - 选择"批量压缩包"
3. **选择文件夹** - 选择包含压缩包的文件夹
4. **编辑文件列表** - 可选择性编辑每个压缩包的文件顺序
5. **批量导入** - 点击 ✅ 按钮批量保存所有书籍

### 📚 管理你的书库

#### 创建收藏夹
1. **进入收藏夹管理** - 在书籍列表页面，点击右上角三点菜单 → "收藏夹管理"
2. **新建收藏夹** - 点击 ➕ 按钮，输入名称、选择图标和颜色
3. **添加书籍到收藏夹** - 长按书籍选择"加入到收藏夹"，或使用批量操作

#### 使用标签系统
1. **创建标签** - 在"书签管理"页面创建不同颜色的标签
2. **为书籍添加标签** - 在书籍操作菜单中选择"添加书签"
3. **按标签筛选** - 在书籍列表使用筛选功能按标签查看书籍

#### 切换显示布局
- 在书籍列表页面，点击筛选按钮可以切换列表视图和网格视图

### 📖 个性化阅读设置

#### 设置阅读方向
1. **打开阅读设置** - 在阅读页面点击右上角设置图标
2. **选择阅读方向**：
   - **从左往右**：适合大多数书籍和漫画
   - **从右往左**：适合日式漫画
   - **从上往下**：适合长条漫画，图片连续滚动显示
3. **设置自动保存** - 选择后设置立即保存，下次自动应用

#### 阅读进度记录
- 应用自动记录每本书的阅读位置
- 重新打开书籍时自动跳转到上次阅读位置
- 在书籍列表中可看到每本书的阅读进度条

## 📖 阅读体验

### 🎯 智能进度显示
- **分页模式**（左右阅读）：显示当前页数和可拖拽的进度滑块
- **滚动模式**（上下阅读）：显示百分比进度和线性进度条
- **实时同步**：阅读进度实时保存到本地数据库

### 🔄 多样化阅读方式
- **传统翻页**：左右滑动翻页，适合漫画和图册
- **连续滚动**：上下滑动，图片无缝连接，适合长条漫画
- **反向阅读**：从右到左翻页，支持日式漫画阅读习惯

### 🖼️ 图片查看增强
- **缩放支持**：双指缩放查看细节（0.5x - 4x）
- **全屏沉浸**：点击切换进度条显示
- **高质量渲染**：保持图片原始质量

- **WebView** - 网页解析

- **Background Downloader** - 后台下载
